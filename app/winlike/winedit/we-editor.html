Title

Label of the window in the titlebar. You may use HTML tags (cf. 49_develop_title.html)

Skin

Layout setting. Defines the sub folder of /skins, in which the layout information (pictures and wm-window.html) for the window class is situated (e.g. default or light).

Content

Relative URL of the content file that should be displayed in the window (e.g. samples/test.html).

Name

The target name of the window. A link with that name as target value will open the referred page in a window of that window class with the corresponding properties such as the skin, size etc. (see also 02_tutorial_content.html and 41_develop_links.html)

In front

Defines whether the window is always in the foreground.

Window visibility

Visibility of the whole window. Invisible windows are not created when calling the website. They will only be visible when content is loaded into this window and the window is being created. In these editor these windows can also be viewed through the window list.

Titlebar visibility

Visibility of the titlebar. If the title bar is invisible, the window can be moved by the status bar at the bottom.

Min/Max visibility

Visibility of the minimize/maximize icon of the titlebar

Close icon visibility

Visibility of the close icons of the titlebar. If it is invisible, the user cannot close the window.

Moveable

Sets or checks whether the window is moveable or not.

Resizeable

Sets whether the window can be enlarged or reduced with the size grip at the right corner at the bottom of the window.

See Properties API for further window properties.
